    진행 중

1.0.0 버전 / 2023년 4월 17일 업로드

토글은 두 가지 상태(예/아니오)를 빠르게 전환하고 즉각적인 결과가 나타나야 하는 상황에서 사용할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드
  • JS 파일 다운로드

구성 요소

토글은 알약 형태의 토글 입력과 그 왼쪽에 배치된 토글 레이블, 오른쪽에 배치된 상태 레이블로 구성됩니다

1. 토글 레이블: 옵션에 대한 정보를 나타냅니다.
2. 토글 입력: 토글의 선택 여부를 나타냅니다.
3. 상태 레이블(선택 사항): 토글의 동작 상태를 설명합니다.

사용 가이드

1. 일반

- 토글을 켜거나 끄면 즉각적으로 그 결과가 나타나야 합니다. - 변경 사항을 적용하기 위해 추가 단계(ex. ‘확인’ 버튼을 클릭해야 하는 경우)에는 체크박스를 사용합니다. - 옵션의 순서는 논리적(예. 가장 자주 조정되는 옵션 순)으로 설정합니다. - 켬/끔 중 어느 것을 기본값으로 하여도 무방하나, 논리적으로 설정합니다.

2. 레이블

- 항상 명확하고 간결한 레이블을 사용하십시오.
- 레이블은 단어나 구를 사용하며, 세 단어 미만으로 가능한 짧게 작성합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.
- 레이블에 말줄임표를 사용하지 마십시오. 불가피하게 긴 레이블을 사용해야 하는 경우 줄바꿈을 허용합니다.
- 더 구체적인 레이블이 있는 경우에만 켬/끔 상태 레이블을 교체할 수 있습니다.
(ex. 레이블이 ‘이미지 보기’인 경우 보기/숨기기 상태 레이블을 사용할 수 있습니다.)

  • 상태 레이블 상시 유지

  • 임의로 레이블 레이아웃 변경 불가

간격 가이드

토글 입력 버튼은 28x18 사이즈를 가지며 입력 버튼과 상태 레이블 사이 간격은 8px입니다.
토글 레이블의 가로 길이는 변동되는 길이이며 토글 레이블과 토글 입력 버튼 사이 간격은 4px입니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 비활성화 상태인 Disabled 상태를 지원합니다.
