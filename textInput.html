Text Input
  • 기획/디자인

    진행 중

  • 코딩

    진행 중

1.0.0 버전 / 2023년 4월 21일 업로드

텍스트 입력을 통해 사용자는 텍스트를 입력하고 편집할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

텍스트 입력은 박스 형태의 텍스트 입력 필드와 그 왼쪽 또는 위에 배치된 텍스트 입력 레이블, 밑에 배치된 도움말로 구성됩니다

1. 텍스트 입력 레이블: 입력해야 하는 내용(텍스트)에 대한 정보를 나타냅니다.
2. 텍스트 입력 값: 사용자가 텍스트 필드에 입력한 내용(입력 텍스트)입니다. 텍스트를 입력하지 않았을 때는 자리 표시자가 나타날 수 있습니다.
3. 텍스트 입력 필드: 사용자가 내용(텍스트)를 입력하는 컨테이너입니다.
4. 도움말(선택 사항): 사용자가 올바른 입력 값을 작성하도록 추가적인 정보를 제공할 때 사용하는 보조 레이블입니다.

사용 가이드

1. 일반

- 텍스트 입력 필드의 너비는 예상되는 입력 값의 길이를 반영하여 설정합니다.
- 레이블이 수평으로 배치된 텍스트 입력과 수직으로 배치된 텍스트 입력을 함께 사용하지 않습니다.
- 아무것도 입력되지 않은 상태를 기본값으로 하지만, 입력값을 사전에 알 수 있는 예외적인 경우(예. 최근 배송지)에 한하여 예측값을 기본값으로 할 수 있습니다.

2. 레이블

- 레이블은 단어나 구를 사용하며, 세 단어 이하로 간결하게 작성합니다.
- 필수 항목에 대해서는 레이블 뒤에 ‘*’(red 60)를 붙혀 필수임을 인지할 수 있도록 합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.
- 경우에 따라 레이블이 포함되지 않을 수 있습니다. 그때에는 레이블을 텍스트 필드 안 입력 보조 텍스트를 대체하여 사용해주세요.

3. 자리표시자

- 입력 값에 대한 예시 또는 힌트를 제공합니다.
- 사용자에게 도움이 되는 경우에 한해서 자리 표시자를 제공합니다.

4. 도움말(선택 사항)

- 구두점이 있는 완전한 문장으로 작성합니다.

  • 같은 스타일 사용

  • 스타일 병용하여 사용 불가

  • 텍스트 영역으로 사용 가능

  • 비밀번호 스타일을 지원

간격 가이드

레이블과 입력 버튼 사이 간격은 8px. 입력 버튼과 도우미 텍스트 사이 간격 또한 8px을 사용합니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 읽기 전용 상태인 Read only, 에러 상태인 Error, 비활성화 상태인 Disabled 상태를 지원합니다.
