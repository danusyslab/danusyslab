    진행 중

1.0.0 버전 / 2023년 6월 5일 업로드

테이블은 정보를 나열하여 전달하는 컨테이너 입니다. 테이블을 통해 유저가 빠르고 쉽게 정보를 인지하고 분석할 수 있도록 합니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

테이블은 헤더와 바디로 구성됩니다.

1. 헤더 컨테이너 : 테이블 칼럼의 헤더로 구성된 컨테이너 입니다.
2. 헤어 텍스트 : 테이블 헤더 텍스트 입니다.
3. 바디 컨테이너 : 테이블 칼럼의 바디로 구성된 컨테이너 입니다.
4. 바디 텍스트 : 테이블 바디 텍스트 입니다.

사용 가이드

- 정렬 된 데이터 정보를 보여줄 때 사용합니다.
- 페이지네이션을 포함할 수 있습니다.
- 종류, 검색, 필터링 된 데이터를 표출할 수 있습니다.
- 헤더 컨테이너를 고정형으로 사용할 수 있습니다.

  • 테이블만 사용 가능

  • 페이지네이션과 함께 사용 가능

간격 가이드

테이블 텍스트는 각 칼럼에서 가운데 정렬, 테이블과 페이지네이션 사이 간격은 16px을 사용합니다.

상태

기본 스타일인 Default 와 마우스를 올린 상태인 Hover 상태를 지원합니다.

테마

Light 모드와 Dark 모드를 지원합니다.

종류

Default 와 같은 스타일 종류로 Compact 스타일을 지원합니다.
