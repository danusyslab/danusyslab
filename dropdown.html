    진행 중

1.0.0 버전 / 2023년 4월 26일 업로드

텍스트 입력을 통해 사용자는 텍스트를 입력하고 데이터를 검색할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드
  • JS 파일 다운로드

구성 요소

드롭다운은 박스 형태의 드롭다운 입력 필드와 그 왼쪽 또는 위에 배치된 드롭다운 레이블, 밑에 배치된 도움말로 구성되며, 입력 필드 클릭 시 옵션 목록이 나타납니다.

1. 드롭다운 레이블: 선택해야 하는 내용(옵션)에 대한 정보를 나타냅니다.
2. 드롭다운 입력 값: 사용자가 드롭다운에 입력한 내용(선택 옵션)입니다. 아무것도 선택되지 않았을 때는 자리 표시자가 나타날 수 있습니다.
3. 드롭다운 아이콘: 이 입력 필드가 드롭다운임을 암시하며, 메뉴가 열린 상태인지 닫힌 상태인지를 알려줍니다.
4. 드롭다운 입력 필드: 드롭다운 입력 값과 아이콘을 포함하는 컨테이너입니다.
5. 옵션: 사용자가 선택할 수 있는 항목입니다. 드롭다운 유형에 따라 하나 또는 복수의 옵션을 선택할 수 있습니다.
6. 옵션 레이블: 해당 옵션에 대한 정보를 나타냅니다.
7. 옵션 체크박스: 복수의 옵션을 선택할 수 있는 경우에 나타납니다.
8. 메뉴: 선택할 수 있는 옵션들의 목록으로, 열고 닫을 수 있습니다.

사용 가이드

1. 일반

- 드롭다운 입력 필드의 너비는 예상되는 입력 값의 길이를 반영하여 설정합니다.
- 레이블이 수평으로 배치된 드롭다운과 수직으로 배치된 드롭다운을 함께 사용하지 않습니다.
- 아무것도 입력되지 않은 상태를 기본값으로 하는 것을 권장하나, 대부분의 사용자가 선택할 것으로 예측되는 항목이 있는 경우에 한하여 예측값을 기본값으로 할 수 있습니다.

2. 레이블

- 레이블은 간결하고 명확하게 작성합니다.
- 필수 항목에 대해서는 레이블 뒤에 ‘*’를 붙혀 필수임을 인지할 수 있도록 합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.
- 경우에 따라 레이블이 포함되지 않을 수 있습니다. 그때에는 레이블을 텍스트 필드 안 입력 보조 텍스트를 대체하여 사용해주세요.

3. 자리표시자

- 입력 값에 대한 예시 또는 힌트를 제공합니다.
- 사용자에게 도움이 되는 경우에 한해서 자리 표시자를 제공합니다.

4. 메뉴/옵션

- 드롭다운이 화면에 나타나는 위치에 따라 메뉴 드롭다운 상단 또는 하단에 위치할 수 있습니다.
- 모든 옵션을 동일한 품사 또는 동사 시제를 사용하여 표현합니다.
- 옵션 레이블은 간결하고 명확하게 작성합니다.
- ‘선택하지 않음’ 또는 ‘없음’ 옵션을 포함할 수 있습니다.
- 옵션 개수가 많아(6개 이상) 스크롤 되어야 하는 경우, 스크롤을 생성하여 다른 옵션들이 더 있음을 알립니다.
- 메뉴를 열 때, 현재 선택한 옵션이 있으며 메뉴가 스크롤이 가능한 경우에는 선택된 옵션이 메뉴 상단에 보일 수 있도록 메뉴가 스크롤이 된 상태로 열립니다.
- 옵션의 순서는 가나다순으로 제공하는 것을 권장합니다.

  • 같은 스타일 사용

  • 스타일 병용하여 사용 불가

간격 가이드

레이블과 입력 버튼 사이 간격은 8px이며 컨테이너 좌우 패딩 값은 12px 입니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus 상태를 지원합니다.
