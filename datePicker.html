Date picker
  • 기획/디자인

    진행 중

  • 코딩

    시작 안함

1.0.0 버전 / 2023년 5월 10일 업로드

데이트 픽커를 사용하여 특정 날짜를 선택할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

데이트 픽커는 박스 형태의 데이트 픽커 입력 필드와 그 왼쪽 또는 위에 배치된 드롭다운 레이블, 밑에 배치된 도움말로 구성되며, 입력 필드 클릭 시 달력이 나타날 수 있습니다.

1. 데이트 픽커 레이블: 선택해야 하는 내용(옵션)에 대한 정보를 나타냅니다.
2. 데이트 픽커 입력 값: 사용자가 데이트 픽커에 입력한 내용(선택 날짜)입니다. 날짜를 직접 타이핑하여 또는 달력에서 날짜를 선택하여 입력할 수 있습니다. 아무것도 선택되지 않았을 때는 자리 표시자(yyyy. mm. dd.)가 나타날 수 있습니다.
3. 데이트 픽커 입력 필드: 데이트 픽커 입력 값과 아이콘을 포함하는 컨테이너입니다.
4. 도움말(선택 사항): 사용자가 올바른 입력 값을 작성하도록 추가적인 정보를 제공할 때 사용하는 보조 레이블입니다.
5. 선택된 연/월: 현재 선택되어 있는 연도와 달을 나타냅니다.
6. 이전 또는 다음 월: 이전 달 또는 다음 달로 이동합니다.
7. 오늘 날짜: 데이트 픽커를 클릭한 날에는 날을 나타내는 숫자 아래에 2px의 정사각형이 함께 노출됩니다.
8. 날짜: 원하는 날짜를 선택할 수 있습니다.
9. 선택된 날짜: 사용자가 클릭하여 선택된 날에는 원형(danu_blue_60)의 배경색과 함께 흰색으로 날짜가 노출됩니다.
10. 달력: 날짜를 선택하여 입력할 수 있으며, 데이트 픽커 입력 필드를 눌러 열고 닫을 수 있습니다.
11. 이전 또는 다음 월의 날짜: 이전 달 또는 다음 달의 날짜를 나타냅니다.

사용 가이드

1. 일반

- 입력 필드의 너비는 날짜 입력값이 넘치거나 달력의 너비를 넘지 않는 범위 내에서 자유롭게 조정할 수 있습니다.
- 레이블이 수평으로 배치된 데이트 픽커와 수직으로 배치된 데이터 픽커를 함께 사용하지 않습니다.
- 아무것도 입력되지 않은 상태를 기본값으로 하는 것을 권장하나, 대부분의 사용자가 선택할 것으로 예측되는 항목이 있는 경우에 한하여 예측값을 기본값으로 할 수 있습니다.

2. 레이블

- 레이블은 간결하고 명확하게 작성합니다.
- 필수 항목에 대해서는 레이블 뒤에 ‘*’를 붙혀 필수임을 인지할 수 있도록 합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.

3. 자리표시자

- 날짜 표기 양식을 나타냅니다.

4. 도움말(선택 사항)

- 날짜 표기 양식을 나타냅니다.

5. 달력

- 달력의 높이와 너비는 조정할 수 없습니다.
- 달력은 입력 필드와 왼쪽 정렬해야 합니다.
- 달력이 화면에 나타나는 위치에 따라 메뉴 드롭다운 상단 또는 하단에 위치할 수 있습니다.

  • 레이블 가로 배치 사용 가능

  • 레이블 세로 배치 사용 가능

간격 가이드

레이블과 입력 버튼 사이 간격은 8px. 입력 버튼과 도우미 텍스트 사이 간격 또한 8px을 사용합니다.
달력의 상하 좌우 마진은 16px을 사용하며 현재 달, 연도 사이 간격과 날짜 사이 간격은 12px, 날짜들의 사이 간격은 8px을 사용합니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 읽기 전용 상태인 Read only, 에러 상태인 Eroor를 지원합니다.

종류

날짜를 선택할 수 있는 Day picker와 달을 선택할 수 있는 Month picker를 지원합니다.
