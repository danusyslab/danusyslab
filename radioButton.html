Radio Button
  • 기획/디자인

    진행 중

  • 코딩

    진행 중

1.0.0 버전 / 2023년 4월 4일 업로드

라디오 버튼은 두 개 이상의 옵션 중에서 단 하나의 항목을 선택해야 할 때 사용됩니다. 복수의 항목을 선택해야할 경우, 라디오 버튼 대신 체크박스를 사용합니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

라디오 버튼은 원 형태의 라디오 버튼 입력과 그 오른쪽에 배치된 라디오 버튼 레이블로 구성됩니다. 필요한 경우 그룹 레이블을 추가할 수 있습니다.

1. 그룹 레이블(선택 사항): 옵션 그룹의 범주나 선택 지침에 대해서 설명합니다.
2. 라디오 버튼 입력: 라디오 버튼의 선택 여부를 나타냅니다.
3. 라디오 버튼 레이블: 옵션에 대한 정보를 나타냅니다.

사용 가이드

1. 일반

- 사용 가능한 모든 옵션을 노출합니다.
- 또한 옵션은 포괄적이면서도 각 옵션이 명확하게 구분되어야 합니다. 즉 MECE(Mutually Exclusive, Collectively Exhaustive)해야 합니다.
- 옵션의 순서는 논리적(예. 가장 일반적으로 선택되는 옵션 순, 가장 안전한 옵션 순)으로 설정합니다.
- 기본값으로 첫 번째 옵션이 선택되어 있는 상태를 권장하며, 그렇지 않은 경우 미선택된 상태를 기본값으로 합니다.
- 가능한 경우 선택하지 않을 수 있는 옵션(“선택하지 않음”)을 제공하는 것을 권장합니다.
- 라디오 버튼에는 부모-자손 관계가 없습니다.

2. 레이블

- 항상 명확하고 간결한 레이블을 사용하십시오.
- 레이블은 단어나 구를 사용하며, 세 단어 미만으로 가능한 짧게 작성합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.
- 레이블에 말줄임표를 사용하지 마십시오. 불가피하게 긴 레이블을 사용해야 하는 경우 줄바꿈을 허용합니다.
- 레이블이 줄바꿈되는 경우 선택자를 레이블의 첫 줄을 기준으로 정렬하십시오.

  • 단일 선택에만 사용 가능

  • 다중 선택에는 사용 불가능

  • 간결하고 명확한 레이블 사용 가능

  • 불명확하고 긴 레이블은 사용 불가능

  • 줄바꿈 되는 경우 첫 줄을 기준으로 정렬 가능

  • 가운데 정렬 불가능

간격 가이드

입력 버튼은 18px 사이즈를 가지며 입력 버튼과 레이블 사이 간격은 8px입니다.
라디오 버튼은 상하, 좌우 2px의 패딩 값을 가집니다.
세로로 배치할 경우 위아래 여백 4px, 가로로 배치할 경우 좌우 여백 12px를 주어 간격을 유지합니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 읽기 전용 상태인 Read-only, 비활성화 상태인 Disabled 상태, 잘못된 상태인 Error 상태를 지원합니다.
