1.0.0 버전 / 2023년 5월 22일 업로드

버튼 그룹을 통해 버튼을 한 그룹으로 묶어 제공합니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

버튼 그룹은 선택되지 않은 Default 버튼 상태와 선택된 Active 버튼 상태로 구성됩니다.

  • 1. 선택되지 않은 버튼 : 사용자가 선택하지 않은 옵션 값을 나타내는 버튼입니다.
    2. 선택된 버튼 : 사용자가 선택한 옵션 값을 나타내는 버튼입니다.

사용 가이드

1. 일반

- 서로 다른 다양한 기간을 선택하는 옵션 버튼에 사용할 수 있습니다. (ex. 1개월, 3개월, 6개월)
- 서로 다른 다양한 기간 단위를 선택하는 옵션 버튼에 사용할 수 있습니다. (ex. 시, 일, 주, 월)
- 3개 이상의 옵션 값을 제공하는 단일 선택 버튼에 사용할 수 있습니다.

  • 기간 선택 옵션 버튼에 사용 가능

  • 일반 옵션 버튼에 사용 불가능

  • 단일 선택 가능

  • 다중 선택 불가능

스타일

기존의 버튼을 옵션 갯수에 맞게 붙혀서 사용하며 버튼 그룹을 하나로 생각하여 상하, 좌우에 4px의 둥글기를 넣습니다.
기본 버튼 그룹 : 기본적으로 Solid 버튼 스타일로 동일하게 버튼 그룹을 배치하여 사용할 수 있습니다. 이떄 선택된 버튼의 값은 Active 버튼 상태 값을 적용하여 사용합니다.
혼합 버튼 그룹 : Outlined 스타일과 Solid 스타일 버튼을 혼합하여 사용하는 버튼 그룹 스타일입니다. 이때 선택된 버튼의 스타일은 반드시 Solid 스타일로 적용하여 사용합니다.
