    진행 중

1.0.0 버전 / 2023년 4월 24일 업로드

텍스트 입력을 통해 사용자는 텍스트를 입력하고 데이터를 검색할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

검색창은 박스 형태의 텍스트 입력 필드와 선행 아이콘으로 구성됩니다.

1. 선행 아이콘: 검색창의 정체성을 나타내는 검색 아이콘을 배치합니다.
2. 텍스트 입력 값: 사용자가 텍스트 필드에 입력한 내용(입력 텍스트)입니다. 텍스트를 입력하지 않았을 때는 자리 표시자가 나타날 수 있습니다.
3. 텍스트 입력 필드: 사용자가 검색 내용을 입력하는 컨테이너입니다.

사용 가이드

검색 필드 내에 사용자가 텍스트를 입력하지 않은 경우, 해당 자리에 검색에 대한 사용자 컨텍스트를 줄 수 있는 자리 표시자 텍스트를 배치합니다.
검색 필드에 텍스트를 입력 후, 보편적으로 엔터 키를 누르면 검색이 되나, 때에 따라 우측에 별도의 검색 버튼을 둘 수 있습니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus를 지원합니다.
