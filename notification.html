1.0.0 버전 / 2023년 5월 24일 업로드

사용자 경험 중 중요한 정보에 대해서 토스트 혹은 모달창을 띄워 유저에게 전달할 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

토스트

현재 상태에 대해 가시적으로 보여줄 수 있는 아이콘과 메세지 내용, 닫기 버튼으로 토스트를 구성할 수 있습니다.

1. 선행 아이콘 : 현재 상태에 대해 가시적으로 보여줄 수 있는 아이콘입니다.
2. 메세지 내용 : 해당 알람을 통해 전달하려는 텍스트 메세지 입니다.
3. 닫기 버튼 : 누르면 해당 알림이 즉각적으로 닫히는 닫기 버튼입니다.

사용 가이드

- 사용자에게 알릴 메세지나 시스템 상태 상의 변화가 있을 때 사용합니다.
- 토스트의 경우 닫기 버튼을 함께 사용할 수도 있고, 토스트 버튼이 즉각적으로(3초 내외) 사라지는 경우 닫기 버튼을 제외하고 사용할 수 있습니다.
- 토스트를 통해 다른 페이지로 이동 기능을 사용할 수 있으며 이때는 메세지와 닫기 버튼 사이에 버튼을 병용하여 사용합니다.
- 알림 상태에 따라 에러, 경고, 성공, 정보로 사용할 수 있습니다.

  • 즉각적으로 사라지지 않는 경우, 닫기 버튼을 함께 사용

  • 즉각적으로 사라지는 경우, 닫기 버튼을 제외하고 사용 가능

간격 가이드

좌측에 8px의 상태 컬러를 적용한 상태 바와 상하, 좌우에 16px의 패딩 값을 줍니다.

상태

에러 알람, 경고 알람, 정보 알람, 성공 알람을 지원합니다.

스타일

기본 스타일은 닫기 버튼까지 포함된 상태이며 상황에 따라 닫기 버튼을 제외하고 사용할 수 있고 사이에 버튼을 병용하여 사용할 수 있습니다.

모달창

사용자 행동에 대한 피드백을 모달창을 통해 전달할 수 있습니다.

1. 모달 제목(선택 사항) : 모달창의 제목입니다.
2. 메세지 내용 : 해당 알람을 통해 전달하려는 텍스트 메세지 입니다.
3. 버튼 영역 : 취소, 닫기, 확인 등의 버튼을 두어 사용자와 소통할 수 있습니다.

사용 가이드

- 사용자의 행동에 대해 피드백을 주거나 시스템 상태의 변화를 알리기 위해 사용합니다.
- 사용자에게 버튼 클릭을 유도하여 보다 확실한 소통이 필요할 때 사용합니다.
- 모달창의 제목 영역은 선택적으로 사용할 수도 있고 사용하지 않을 수 있습니다.

  • 제목 영역을 함께 사용 가능

  • 제목 영역을 제외하여 사용 가능

간격 가이드

상하, 좌우에 28px의 패딩 값을 주고 제목 영역과 메세지 영역 간의 간격 또한 28px을 줍니다. 메세지 영역과 버튼 영역 사이 간격, 버튼 끼리의 간격은 16px 입니다.
