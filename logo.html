Wordmark 1.0.0 버전 / 2023년 3월 31일 업로드
디바이스 환경 상의 가독성과 출력 시 일관성 유지 등의 제반 사정을 고려하여 로고는 '워드마크' 타입으로 사용합니다.
정해진 스타일에서 벗어난 '워드마크' 사용을 금지하며 형태, 비율 등의 변형 또한 허용하지 않습니다.
Key color: $danu_blue_60
Text type: Campton
Key color: $gray_110
Text type: Campton
Key color: $danu_blue_30
Text type: Campton
Key color: $white
Text type: Campton

워드마크는 단독으로 사용할 것

워드마크로써 로고를 활용할 때는 반드시 주변 텍스트가 없는 안전 여백이 확보되어야 합니다.

보조 텍스트는 로고 타입과 맞춰 사용할 것

워드마크 옆에 보조 텍스트를 넣는 경우에는 1) 프로젝트 타이틀이거나, 2) 다누시스 제품명일 때만 사용 가능합니다.
폰트 타입은 워드마크와 같은 'Campton English'만 사용 가능하며, 다누시스 워드마크와의 차별성을 위해 'Bold'가 아닌 'Regular'만 사용 가능합니다.

정해진 컬러로만 사용할 것

밝은 색상의 배경 위에 사용할 경우 1) $danu_blue_60, 2) $gray_110 컬러만 사용 가능하며, 어두운 색상의 배경 위에 사용할 경우 1) $danu_blue_30, 2) $white 컬러만 사용 가능합니다.
상단의 로고 타입에서 워드마크 타입 로고를 다운 받아 사용해주세요.
