    진행 중

1.0.0 버전 / 2023년 4월 18일 업로드

항목을 설명하는 키워드가 포함된 알약 형태의 컴포넌트입니다. 항목에 레이블을 지정하고 분류할 때 주로 사용됩니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

태그는 알약 형태의 컨테이너와 그 안에 배치된 텍스트 레이블로 구성됩니다.

1. 컨테이너: 태그 레이블을 감싸고 있으며 태그의 영역과 형태를 결정합니다.
2. 태그 레이블: 해당 태그에 대한 정보를 나타냅니다.

사용 가이드

1. 일반

- 솔리드 태그와 아웃라인 태그를 함께 사용하지 않습니다.
- 여러 개의 태그가 함께 사용되는 경우에그 순서는 논리적(예. 가장 중요한 내용 순)으로 설정합니다.
- 태그 색상 또한 논리적으로 결정합니다.(예. 긴급한 정보나 상태는 붉은색 계열 사용)

2. 레이블

- 항상 명확하고 간결한 레이블을 사용하십시오.
- 레이블은 단어를 사용하며, 세 단어 이하(6글자 이하)로 가능한 짧게 작성합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.

  • 같은 스타일 사용

  • 스타일 병용하여 사용 불가

간격 가이드

컨테이너와 레이블 사이 좌우 간격은 8px, 태그 요소 사이 간격도 8px을 사용합니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 비활성화 상태인 Disabled 상태를 지원합니다.
