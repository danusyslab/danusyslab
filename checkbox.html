    진행 중

1.0.0 버전 / 2023년 4월 15일 업로드

체크박스는 여러 옵션 중에서 원하는 수의 항목을 선택하거나 단일 옵션을 선택 또는 해제하는 상황에서 사용될 수 있습니다.

  • 디자인 파일 바로가기
  • SCSS 파일 다운로드

구성 요소

체크박스는 정사각형 형태의 체크박스 입력과 그 오른쪽에 배치된 체크박스 레이블로 구성됩니다. 필요한 경우 그룹 레이블을 추가할 수 있습니다.

1.그룹 레이블(선택 사항): 옵션 그룹의 범주나 선택 지침에 대해서 설명합니다.
2. 체크박스 입력: 체크박스의 선택 여부를 나타냅니다.
3. 체크박스 레이블: 옵션에 대한 정보를 나타냅니다.

사용 가이드

1. 일반

- 사용 가능한 모든 옵션을 노출하는 것을 권장하나, 옵션의 개수가 많은 경우 일부 옵션을 숨길 수 있습니다.(예. 더 보기)
- 옵션은 포괄적이면서도 각 옵션이 명확하게 구분되어야 합니다. 즉 MECE(Mutually Exclusive, Collectively Exhaustive)해야 합니다.
- 옵션의 순서는 논리적(예. 가장 일반적으로 선택되는 옵션 순, 가장 안전한 옵션 순)으로 설정합니다.
- 체크박스는 부모-자식 관계를 만들 수 있습니다.
- 입력 양식이나 필터링, 데이터 테이블, 이용약관 동의, 하위 선택 항목이 있는 경우에 주로 사용합니다.
- 단일 항목을 선택해야 하는 경우 체크박스 대신 라디오 버튼을 사용합니다.
- 체크박스는 선택 또는 해제할 수 있으며, 어떤 체크박스를 선택하거나 선택 해제하더라도 다른 체크박스에 영향을 미치지 않습니다.

2. 레이블

- 항상 명확하고 간결한 레이블을 사용하십시오.
- 레이블은 단어나 구를 사용하며, 세 단어 미만으로 가능한 짧게 작성합니다.
- 레이블에 긍정적이고 적극적인 표현을 사용합니다.
- 레이블에 말줄임표를 사용하지 마십시오. 불가피하게 긴 레이블을 사용해야 하는 경우 줄바꿈을 허용합니다.
- 레이블이 줄바꿈되는 경우 선택자를 레이블의 첫 줄을 기준으로 정렬하십시오.

  • 다중 선택에 적극 사용

  • 단일 선택에는 사용 지양

  • 줄바꿈 되는 경우 첫 줄을 기준으로 정렬 가능

  • 가운데 정렬 불가능

부모 자식 관계

부모 자식 관계를 사용할 수 있습니다. 부모 자식 관계 구조에서 체크된 부모에는 ‘Indeterminate’ 스타일을 적용하여 부모와 자식 간의 스타일을 구분합니다.

간격 가이드

입력 버튼은 16px 사이즈를 가지며 입력 버튼과 레이블 사이 간격은 8px입니다.

정렬 가이드

세로 배치와 가로 배치를 기본으로 사용하며 가로 배치와 세로 배치를 병용하여 사용해야하는 경우에 격자 배치를 사용합니다.

버튼 상태

기본 상태인 Default, 마우스로 클릭한 상태인 Focus, 비활성화 상태인 Disabled 상태, 잘못된 상태인 Error 상태를 지원합니다.
