Space

공간 시스템에서 크기나 길이를 결정하는 기본 단위는 8px입니다. 미세 조정이 필요한 경우(ex. 텍스트 사이 공간)에 한해서는 2px, 4px 단위를 사용할 수 있습니다. 홀수의 사이즈를 허용하지 않습니다.

Shadows

직사광선에 의해 빛의 경로에 따라 생기는 Hard Shadow와 주변광에 의해 빛의 경로와 무관하게 물질 주변에 퍼지는 Soft Shadow로 나뉘어집니다. 자연스러운 그림자를 형성하기 위해 Hard Shadow와 Soft Shadow를 조합하여 사용합니다. 그 생성 규칙은 물질이 상승한 높이가 n px라고 할 때 다음과 같습니다.

Hard Shadow
X = 0 | Y = n | Blur = 3 * n / 2 | Opacity = 32%
Soft Shadow
X = 0 | Y = 0 | Blur = 4n | Opacity = 16%
Mixed Shadow
X = 0 | Y = n | Blur = 4n | Opacity = 32%

Grid system

디바이스는 모바일, 태블릿, 데스크톱을 큰 기준으로 구분하며 그리드는 4의 배수로 분할하여 사용합니다.

Gutters

각 칼럼 사이에 위치하는 패딩 값을 ‘거터’라고 합니다. 각 칼럼 마다 정해진 거터 값에 따라 사용합니다.

Gutters

각 칼럼 사이에 위치하는 패딩 값을 ‘거터’라고 합니다. 각 칼럼 마다 정해진 거터 값에 따라 사용합니다.

Margins

마진의 대지의 상하, 좌우에 들어가는 여백을 뜻합니다. 각 대지 마다 정해져 있는 마진 값에 따라 사용합니다.

Margins

마진의 대지의 상하, 좌우에 들어가는 여백을 뜻합니다. 각 대지 마다 정해져 있는 마진 값에 따라 사용합니다.

Columns

컬럼은 각 아이템이 위치하는 영역을 뜻합니다. 16 Grid 기준 col-16 컬럼은 16 컬럼 전체를 차지하며 col-8은 8컬럼을 차지하게 됩니다.
컬럼은 4와 8의 배수를 기준으로 나누어 사용하는 것을 원칙으로 하며 특별한 상황에 따라 2의 배수로 사용할 수도 있습니다. 홀수의 배수로 칼럼을 나눠서 사용할 수는 없습니다.
